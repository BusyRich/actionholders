Action Placeholders

Who wants plain placeholders, kittens, or this Chris guy all over their dev site?
I know I don't. I WANT ACTION!

Usage

http://actionholder.com/i/{size}
http://actionholder.com/i/250

Gets a square image where width and height equal {size}

http://actionholder.com/i/{width}/{height}
http://actionholder.com/i/150/75

Gets an image with a set width and height

http://actionholder.com/i/{image}/{width}/{height}
http://actionholder.com/i/a1/150/75

Gets a specifc image with a set width and height
